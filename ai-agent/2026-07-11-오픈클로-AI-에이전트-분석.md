### Context
오픈클로(OpenClaw)는 기존 'Clawdbot'과 'Moltbot' 프로젝트의 계보를 잇는 오픈 소스 자율 AI 에이전트(Autonomous AI Agent) 프로젝트입니다. 사용자가 직접 개입하지 않아도 복잡한 컴퓨터 작업을 스스로 수행하도록 설계되었으며, 로컬 환경에서 구동 가능한 '행동형(Agentic) AI'를 지향합니다.

### Core
오픈클로는 사용자의 명령을 받아 OS 레벨의 제어, 브라우저 자동화, 데이터 수집 등의 작업을 처리합니다. 핵심 작동 구조는 다음과 같습니다.

*   **LLM 연동**: 클로드(Claude), GPT, 제미나이(Gemini) 등의 외부 대형 언어 모델(LLM) API를 연결하여 지능적 판단 수행
*   **컴퓨터 제어**: 로컬 환경에서 마우스 클릭, 키보드 입력 등 사용자 인터페이스(UI)를 직접 조작
*   **스케줄링**: 크론(Cron) 작업을 통한 주기적인 업무 자동화 가능
*   **인터페이스**: 메신저나 전용 대시보드를 통해 에이전트의 상태를 확인하고 제어 가능

```bash
# 기본 설치 예시 (터미널)
git clone https://github.com/openclaw/openclaw
cd openclaw
npm install
npm run start
```

### Insight
오픈클로는 인간의 수동적인 명령 입력을 넘어, 에이전트가 능동적으로 목표를 달성하는 '에이전틱 AI(Agentic AI)' 시대를 잘 보여주는 사례입니다. 로컬 설치형 오픈 소스이므로 데이터 보안 측면에서 강점이 있으나, API 연동 시 발생하는 토큰 비용 관리와 외부 명령 수행 시의 보안 위험(Security Risk)은 사용자가 직접 고려해야 할 숙제입니다. 단순 반복 업무를 넘어서는 고도의 복합 작업 자동화에 최적화된 도구로 판단됩니다.
